# Unix Shell Script
Unix/Linux Shell Programming

## Syntax: 
```
#!/bin/bash
#Write your program here.
echo "Hello World !"
```

## Change to executable mode: 
```
$chmod 775 <file name>
$chmod 775 scripts.sh
or
$chmod +x <file name>
$chmod +x scripts.sh
```

## To Run: 
```
$<file path>
$./scripts.sh
```
